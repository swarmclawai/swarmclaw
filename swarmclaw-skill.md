# SwarmClaw ClawHub Skill

The `swarmclaw` skill lets OpenClaw agents manage a SwarmClaw instance via CLI from chat.

## What It Is For

- Dispatching work to specific SwarmClaw agents
- Checking fleet status (agents + active sessions)
- Running diagnostics when a SwarmClaw instance is unhealthy
- Coordinating multi-agent workflows through tasks instead of manual command chains

## Install

```bash
clawhub install swarmclaw
```
